# My first prompt

A place to keep prompts that worked. Example:

```
Read ../../starter-kit/sample-data.csv and tell me the three most
surprising things in it. Then make an HTML dashboard of the highlights
and save it as dashboard.html in this folder.
```

What I learned: Claude Code can read a file two folders up with `../../`, and it writes the HTML right next to me so I can open it in the browser.
