# Starter prompts

Copy any of these into Claude Code while you're in your own folder. Change the details to make them yours.

## Analyze the sample data
```
Read ../../starter-kit/sample-data.csv. Give me a summary, then tell me the
three most surprising patterns. Which application channel actually leads to
interviews and offers? Save your findings as findings.md in my folder.
```

## Turn the data into a dashboard
```
Using ../../starter-kit/sample-data.csv, build a clean HTML dashboard with
summary stats at the top and the key findings highlighted. Save it as
dashboard.html in my folder so I can open it in my browser.
```

## Tailor the resume to the job
```
Read ../../starter-kit/resume-example.md and
../../starter-kit/job-description-example.md. Tailor the resume to that job:
match the language, lead with the most relevant wins, keep it to one page.
Save it as tailored-resume.md in my folder.
```

## Build a one-page portfolio
```
Build me a clean, modern one-page portfolio website as a single HTML file.
Ask me a few questions about my background first, then save it as
portfolio.html in my folder. Make it something I'd be proud to send a
recruiter.
```

## Make a CLAUDE.md about yourself
```
Interview me for about 10 questions to learn who I am and how I work, then
write a CLAUDE.md in my folder so future chats already know me. Keep it
concise and in plain language.
```

## Save your work
```
Save my work: commit everything in my folder and push my branch.
```
